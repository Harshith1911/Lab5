Django Web Application

For successful run of this HelloWorld project, system must have,

1. python version 3.10.2
2. pip version 22.0.3
3. django version 4.0.2

After that,
Open project using any python editor 
Then in the terminal,
Go to the directory where folder HelloWorld is:
then move to the directory HelloWorld>HelloWorld
Now run,
python manage.py runserver

hello world message will be displayed.
